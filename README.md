# Energy-Consumption--rediction
## 📌 Overview
This project predicts electricity consumption using weather and event data.

## 📊 Features
- Synthetic dataset generation
- Data analysis
- Data visualization
- Machine learning model

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📈 Insights
- Energy increases with temperature
- Special events increase consumption
- Seasonal trends observed

## ▶️ How to Run
1. Open Jupyter Notebook
2. Run all cells
3. Use prediction function
